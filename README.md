# PhotoshopChecker

## Задача
Исследовать способы определения подделки/изменения/редактирования изображения с 
помощью Photoshop и реализовать программу, позволяющую это определить.

## Исследованные способы решения
1. Проверка и чтение метаинформации файла
	- Приемущества: простота реализации, низкая алгоритмическая сложность и высокая скорость работы
	- Недостатки: относительно низкая достоверность, т.к. метаинформацию файла можно изменить
2. Проверка на соотвествие с оригиналом
	- Приемущества: сравнительно простая сложность реализации, крайне высокая степень достоверности
	- Недостатки: требует наличие оригинала, в некоторых случаях может работать крайне медленно
3. ELA (Error Level Analysis)
	- Приемущества: сложность подделки и высокая достоверность
	- Недостатки: высокая алгоритмическая сложность и сложность реализации, не работает с фотографиями с контрастным фоном,
	- работает только с форматом JPEG
4. Luminance Gradient Analysis 
	- Приемущества: является одним из самых достоверных способов определения фотошопа, сложно подделать
	- Недостатки: крайне нетривиальная с алгоритмической точки зрения задача

## Решение
В качестве решения поставленной задачи был взят способ на основе проверки метаданных файла. 
Сама реализация состоит из визуальной части (Winforms) позволяющей выбрать изображение и
реализации интерфейса IImageAnalyzer, позволяющей с помощью проверки XMP тегов проверить,
редактировался ли файл в фотошопе.
В данный момент решение может работать с форматами: PNG, JPG, TIF.

## Структура проекта
	- директория ExamplePictures содержит примеры картинок, с помощью которых можно проверить работоспособность.
	- директория Service/Analyzer содержить интерфейс анализа изображения, и его реализация на основе проверки XMP.
	- директория View содержит логику отображения и визуальную часть решения.
